# Example scenes

Each file in this folder is a JSON scene that loads directly into [SVG Animation Studio](https://svg-animation-studio.netlify.app).

To load an example in the app: **Help → Examples → Load as new scene**.

---

## Current examples

| File | Name | Tags | What it demonstrates |
|---|---|---|---|
| `bouncing-ball.json` | Bouncing Ball | motion, beginner, loop | Keyframe animation with squash & stretch |
| `logo-reveal.json` | Logo Reveal | reveal, text, beginner | Staggered two-element entrance sequence |
| `color-pulse.json` | Color Pulse | color, loop, beginner | Ping-pong fill color animation |
| `motion-path.json` | Motion Path | motion, path, intermediate | Circle following a 2D Bezier path |
| `progress-bar.json` | Progress Bar | ui, gradient, intermediate | Gradient scaleX + opacity crossfade |
| `staggered-text-reveal.json` | Staggered Text Reveal | text, reveal, intermediate | Words fading in with stagger delay |
| `loading-spinner.json` | Loading Spinner | ui, loop, beginner | Three-dot bounce with phase offsets |
| `gradient-sunrise.json` | Gradient Sunrise | gradient, motion, intermediate | Radial gradient sun rising on y-track |

## Adding an example

See [CONTRIBUTING.md](../CONTRIBUTING.md) for the full guide and quality checklist.

Short version:
1. Build your scene in the studio
2. Export as Scene JSON
3. Add to this folder with a descriptive kebab-case filename
4. Open a PR using the example template
