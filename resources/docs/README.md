# Documentation

Welcome to the SVG Animation Studio documentation. The following guides cover everything you need to know to create, animate, and export your vector motion projects:

- [Getting Started](getting-started/README.md) — Learn how to launch the studio and create your first animation in 5 minutes.
- [Animation Concepts](animation-concepts/README.md) — Understand tracks, easing, keyframes, loop modes, stagger, and motion paths.
- [AI Prompting Guide](ai-prompting/README.md) — Tips, modes, and example prompts for using the AI-assisted motion features.
- [Export Guide](export-guide/README.md) — A guide to static, SMIL, CSS, JSON, HTML, slide deck, and video exports.
- [Scene JSON Schema Reference](scene-json-schema/README.md) — Technical details of the scene structure for hand-crafting or tooling.
- [Keyboard Shortcuts](keyboard-shortcuts/README.md) — Speed up your design workflow with canvas, timeline, and selection shortcuts.
- [CSS Filter Effects](css-filters/README.md) — Implement glows, blurs, shadows, and glassmorphism in your designs.
